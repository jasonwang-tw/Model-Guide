# 桌面截圖讀取

> **平台限制**：目前截圖路徑僅支援 macOS（`/Users/{username}/Desktop/`）。

找出桌面上最新的截圖並讀取。

## 指令格式

```
/png [數字] [文字敘述（可選）]
```

或

```
/png [文字敘述]
```

## 參數解析規則

- 若第一個參數是數字：讀取該數量的截圖，其餘文字為針對截圖的問題或指令
- 若第一個參數不是數字（或無參數）：預設讀取 1 張最新截圖，所有文字為針對截圖的問題或指令

## 範例

| 指令 | 行為 |
|------|------|
| `/png` | 讀取最新 1 張，描述內容並詢問使用者要做什麼 |
| `/png 3` | 讀取最新 3 張，描述內容並詢問使用者要做什麼 |
| `/png 這張圖是什麼意思？` | 讀取最新 1 張，直接回答問題 |
| `/png 2 這兩張有什麼不同？` | 讀取最新 2 張，直接回答問題 |

## 執行步驟

1. 解析 `$ARGUMENTS`，判斷截圖數量（預設為 1）與文字問題（若有）
2. 用 Bash 執行以下指令找出最新的 N 張圖片：
   ```bash
   ls -t ~/Desktop/*.png ~/Desktop/*.jpg ~/Desktop/*.jpeg 2>/dev/null | head -[數字]
   ```
3. 用 Read 工具依序讀取所有找到的圖片
4. 若有文字問題，直接依照該問題或指令回應
5. 若無文字問題，描述圖片內容，並詢問使用者要對這些截圖做什麼
